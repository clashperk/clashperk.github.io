ClashPerk - Guide

Tracks your clan donation and shows you information about clans, players, clan wars and many more. ClashPerk requires the following permissions to be usable~

  • Read Messages
  • Add Reactions
  • Read Message History
  • Send Messages
  • Embed Links
  • Use External Emojis
  • Attach Files

Commands

Tracker

*start <#tag> [channel] [hexColor] - starts tracking in a channel (requires manage server).

The above command would work as
*start <#tag> [channel]
*start <#tag> [hexColor]
*start <#tag> [hexColor] [channel]
*start <#tag> [channel] [hexColor]

Examples
*start #8QU8J9LP
*start #8QU8J9LP #tracker
*start #8QU8J9LP #5970C1
*start #8QU8J9LP #5970C1 #tracker
*start #8QU8J9LP #tracker #5970C1

*stop <#tag> - stops tracking for a clan (requires manage server).

Example *stop #8QU8J9LP

*stop-all - stops tracking all clans on the server (requires manage server).

Example *stop-all

*edit <#tag> <#hexColor> - edits the color of embed for a clan (requires manage server).

Example *edit #8QU8J9LP #5970C1

*tracking - shows all tracking details of your guild.

Profile

*link-profile <#tag> [optional member] - saves a player to your discord account.

Examples
*link-profile #9Q92C8R20
*link-profile #9Q92C8R20 @Suvajit

*link-clan <#tag> [optional member] - saves a clan to your discord account.

Examples
*link-clan #8QU8J9LP
*link-clan #8QU8J9LP @Suvajit

*profile [optional member] - shows information about your clan.

Examples
*profile
*profile @Suvajit

*my-clan [optional member] - shows information about your clan.

Examples
*my-clan
*my-clan @Suvajit

Lookup

*clan <#tag> - clash of clans clan lookup command.

Example *clan #8QU8J9LP

*player <#tag> - clash of clans player lookup commnad.

Example *player #9Q92C8R20

*units <#tag> - shows troops and spells of the player.

Example *units #9Q92C8R20

*search-clan <#tag> - search clans by name.

Example *search-clan air hounds

*th-compo <#tag> - calculates your clan war weight.

Example *th-compo #8QU8J9LP

*warlog <#tag> - clash of clans war log lookup command.

Example *warlog #8QU8J9LP

*members <#tag> [--th/-th] [th level] - list of clan members.

Examples
*members #8QU8J9LP (list by rank)
*members #8QU8J9LP -th (list by th)
*members #8QU8J9LP -th 10 (shows only th10)
*members #8QU8J9LP -th 12 (shows only th12)

Fun

*facts - displays random clash of clans facts.

*meme - displays random clash of clans memes.

*vote - gives you the link to vote for clashperks.

Config

*config - displays settings of the guild.

*prefix [new prefix] - displays or changes the prefix of the guild.

Examples
*prefix (displays prefix)
*prefix ! (changes prefix)

Requires `Manage Server` to change prefix.

*restrict <member> - restricts or unrestricts someone from using sensitive commands (requires manage server).

Examples
*restrict @Suvajit
*restrict 444432489818357760

Use it again on the same user to remove them from the restriction.

Util

*about - shows information about the bot.

*guide - shows information about how to use bot.

*help - displays a list of commands or information about a command.

*invite - gets the bot invite link.

*ping - pings me!

*stats - displays statistics about the bot.
