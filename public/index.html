Clan Management

In order to use these commands, you need to be a leader / co-leader of the clan.

*cgboard [clanTag]
Admin only
Setup a live updating clan games board
Example
*cgboard #8QU8J9LP
    Things to know..
  • The bot needs following permissions to be usable...
    • Use External Emojis
    • Embed Links
    • Send Messages
  • You can setup multiple clans in a channel.
  • The board updates every 30 min.
*onlineboard [clanTag]
Admin only
Setup a live updating last online board
Example
*onlineboard #8QU8J9LP
    Things to know..
  • The bot needs Use External Emojis and Embed Links permissions.
  • You can setup multiple clans in a channel.
  • The board updates every 90 sec.
*donationlog [clanTag]
Admin only
Setup a donation log in a discord channel
Example
*donationlog #8QU8J9LP
    Things to know..
  • The bot needs Use External Emojis and Embed Links permissions.
  • You can setup multiple clans in a channel.
  • The log updates every 90 sec.
*playerlog [clanTag]
Admin only
Setup a clan member join / leave log in a discord channel
Example
*playerlog #8QU8J9LP
    Things to know..
  • The bot needs Use External Emojis and Embed Links permissions.

*toggle [METHOD] [clanTag]
Admin only

*toggle playerlog #8QU8J9LP

*toggle clangame #8QU8J9LP
*toggle clanembed #8QU8J9LP

*remove [clanTag]

Profile

Link your clan and profile to use commands without any tag
*link [METHOD] [TAG]
Links a clan or player to your discord account
Parameters
METHOD
  • player = links a player to your discord account
  • clan = links a clan to your discord account
TAG
  • CLAN TAG
  • PLAYER TAG
  • no parameters = links a clan to your discord account
  • METHOD = links in-game player or clan to your account
Example
*link #8QU8J9LP
*link clan #8QU8J9LP
*link player #9Q92C8R20
    Things to know..
  • You can only link one clan to your account.
  • But you can link multiple players to your account.
*profile [USER]
Shows information about your profile
Parameters
  • no parameters = shows information about your own profile
  • USER = shows information about the user's profile
Example
*profile
*profile @SUVAJIT
*unlink [PLAYER TAG]
Unlinks a player form your discord account
Example
*unlink #9Q92C8R20
